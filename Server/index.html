系统设计入门

基础概念

Kafka

简单说，也是一种MessageQueue，其提供了消息的订阅与发布的消息队列。用作系统间解耦、异步通信、削峰填谷。

有如下特点：

  • 请求可以被缓存到内存和磁盘中，并且支持事务性，可以通过消息还原相关操作。
  • 支持多个队列
  • 会定期的检查日志文件，然后将过期的数据从 log 中移除。

ZooKeeper

是一个分布式存储系统，能保证多个ZooKeeper之间是强一致的。

其使用场景，可以用来做服务的关联和负载均衡。（其它服务在ZooKeeper中注册自己的ip，另外的服务调用时，可以获取到ip列表，然后随机调用）

Flink

简单说，就是实时计算框架。支持在数据上报过程中，实时的将数据处理出来。 比如双十一，实时查看产品销量、整体收入、用户画像等信息。

RPC

谁能用通俗的语言解释一下什么是 RPC 框架？

目的就是为了像调用本地函数一样调用远端的函数，在分布式系统中使用非常广泛。

POD

就是容器，类似Docker的概念（与Docker并列的）

DDL & DML

  • DDL是表示对数据库表的操作 （如创建表）
  • DML是表示对数据库内容的操作 （如插入数据、删除数据）

K8S

即 Kubernetes，一句括概括：K8S是一个集群管理的工具。

Kubernetes是什么?

Kubernetes和Docker到底有啥关系？

HDFS

是一种分布式的文件系统，主要用来存储大数据量

HBase

是一个分布式、可伸缩的大数据存储，基于HDFS实现的

NGINX

作用：

  • 轻量Web服务器
  • 反向代理
  • 负载均衡
  • Http缓存（即用户访问过的内容，在nginx建立副本，一段时间内再次访问时，可以直接返回给该数据。Nginx 缓存机制详解！）
  • 支持高并发(5W连接)

最简单的HttpSvr

JS

let express = require("express");
let app = express();
 
 // 加载指定目录作为静态页面目录
app.use(express.static("../develop/_book"));
 
app.listen(8080, ()=>{
    console.log("listen 8080");
})

GO

package main

import (
	"log"
	"net/http"
)

func main() {
    // 使用_book作为静态页面目录
	h := http.FileServer(http.Dir("./_book"))
	err := http.ListenAndServe(":9090", h)
	if err != nil {
		log.Fatal("ListenAndServe: ", err)
	}
}
