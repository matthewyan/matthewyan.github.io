调试

VSCode中调试页面

如果下载的源码需要用npm run dev来运行的话：

  • 先用npm run dev跑起页面来
  • 配置工程调试，环境选chrome
  • 编辑launch.json，将url修改为npm run dev后启动的页面url
  • 然后再在VSCode中执行调试，此时下的断点就可以正常断下来了
