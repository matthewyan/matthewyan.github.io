堆

一种特殊的树，是完全二叉树；堆中每一个节点的值都必须大于等于（或小于等于）其子树中每个节点的值。

堆化：当插入新的元素后，堆的条件不再满足时，就需要调整结点，以使其重新满足，这个过程就是堆化。
